# 💻 AI Next Word Predictor

A Deep Learning based AI text generation project built using TensorFlow, LSTM and Gradio.

## 🚀 Features

- Next word prediction
- AI text generation
- Deep Learning model
- Interactive Gradio UI
- Adjustable creativity

## 🛠️ Technologies Used

- Python
- TensorFlow
- LSTM
- Gradio
- NumPy

```bash
